hitTest

  • hitTest(node: HTMLElement, clientX: number, clientY: number): boolean
    • Defined in index.ts:83

    Test whether a client position lies within a node.

    Parameters

    • node: HTMLElement

      The DOM node of interest.

    • clientX: number

      The client X coordinate of interest.

    • clientY: number

      The client Y coordinate of interest.

    Returns boolean

    true if the node covers the position, false otherwise.

    Example

    var obj = document.createElement('img');
obj.style.position = 'absolute';
obj.style.left = 0;
obj.style.top = 0;
obj.width = 100;
obj.height = 100;
document.body.appendChild(obj);

hitTest(obj, 50, 50);   // true
hitTest(obj, 150, 150); // false
